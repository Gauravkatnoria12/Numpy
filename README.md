# Numpy Learning Notes

This repository contains notebooks for learning and practicing NumPy, the fundamental package for numerical computing in Python.

## Notebooks

- `Numpy.ipynb` - Core NumPy basics and introductory examples.
- `Array_Operations.ipynb` - Examples and exercises focused on array creation, manipulation, and operations.
- `Exersises.ipynb` - Practice problems and exercises to reinforce NumPy concepts.

## Purpose

The goal of this repository is to collect and organize NumPy learning materials, examples, and exercises in a single place for study and review.

## Usage

Open the notebooks in Jupyter Notebook or JupyterLab to explore the NumPy examples and run the provided code cells.

## Notes

- This repository is a learning reference and may be updated as new NumPy topics are explored.
- The notebook filenames reflect the main focus of each file.
